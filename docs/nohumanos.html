No Humanos (Non Human) is a crowdfunding project aiming to raise funds for a farm animal shelter.

Main goal:
Raise funds for Finca Arkadia, a farm animal shelter in Tenerife.

How?
I designed, manufactured and sold three t-shirt designs, each one inspired by an inhabitant of the farm.

Results:
The initial fundraising target set by stakeholders was exceeded 2.3 times.
